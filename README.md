# ShootingGame

## 概要

Unity で作っている 2D シューティングゲームです。

- 2D シューティング (現在は縦スクロールのみ)
- 対応プラットフォーム
  - Windows 10  
    ※動作未確認ですが、おそらく Mac でも動くと思います。
- 一人用

## デモ (Screen Shots)

- プレイ画面

## 利用方法

**Windows 10:**

1. Unity プロジェクトをビルド。

1. 作成された .exe ファイルを実行する。

## 開発時 Unity バージョン

- 2020.2.2f1
